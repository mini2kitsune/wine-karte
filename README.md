# 🍷 ワインノート（Wine Notes）

ワインエキスパート学習用のテイスティング記録アプリ（PWA・スマホ完結）。

- **ライブURL**: https://mini2kitsune.github.io/wine-karte/
- **構成**: 単一 `index.html` + `manifest.json` + `sw.js`（Service Worker）
- **データ**: localStorage（端末内キャッシュ）+ Supabase（PostgreSQL / Auth / RLS）で端末跨ぎ同期
- **認証**: Supabase Auth（メール）。ログインしなくても端末内のみで利用可能

## 基本の流れ

1. `+ 記録する` または `⚡`（クイック入力）で新規記録
2. 写真を撮影 or カメラロールから選択
3. ワイン名・生産者・産地・品種・ヴィンテージなどを入力
4. **見た目・香り・味わい・一言メモ** を記入（📚 試験対策チップからの入力も可）
5. 保存すると端末に即保存され、ログイン済みなら Supabase に自動同期

## 📥 メモから取り込み（インボックス）

**飲みながらの記録はメモアプリでOK。清書は不要です。**
一覧画面の「📥 取り込み」から、走り書きメモや AI 添削の返信を貼り付けるだけでカルテが自動作成されます。

推奨ワークフロー:

1. 飲みながら → メモアプリに走り書き（今まで通り）
2. 休日 → アプリの「📥 取り込み」を開き、メモを貼り付け
3. **① 添削依頼文をコピー** → ChatGPT / Claude に貼って送信
   （二次試験講師として添削し、最後に構造化JSONを返すプロンプトが自動付与されます）
4. AIの返信をインボックスに貼り直して **② 解析してカルテを作成**
   → ワイン名・品種・産地・見た目・香り・味わい・添削コメントが各欄に自動で振り分けられます

補足:

- AIを経由せず、メモだけで②を押しても解析されます（ラベル行 `香り: 〜` などを認識。何もなければ先頭行=ワイン名・残り=メモ）
- **元のメモは原文のまま保存**され、カルテ下部の「📥 元のメモ」から読み返せます
- 解析結果はカルテ上でそのまま修正・追記できます

## 📱 ホーム画面に追加（アプリとして使う）

### iOS Safari
1. Safari でライブURLを開く → 共有（□↑）→「ホーム画面に追加」

### Android Chrome
1. Chrome で開く → 「⋮」→「アプリをインストール」

アイコンタップで全画面起動。オフラインでも起動・閲覧できます。

## データベース（Supabase）

- スキーマ: `supabase_setup.sql`（初期構築用）
- マイグレーション: `supabase_migration_raw_memo.sql`
  （📥 取り込み機能の `raw_memo` カラム追加。**SQL Editor で1回実行してください**）
- RLS 有効: 各ユーザーは自分の `wine_notes` 行のみ読み書き可能
