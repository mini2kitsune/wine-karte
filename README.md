# 🍷 ワインカルテ

ワインエキスパート学習用のテイスティング記録アプリ（スマホ完結）。

## 使い方

### 基本の流れ

1. `+` ボタンで新規記録を開始
2. 写真を撮影 or カメラロールから選択
3. ワイン名・生産者・産地・品種・ヴィンテージ・購入場所・飲んだ日を入力
4. ワインタイプ（赤/白/ロゼ/スパークリング/デザート）を選択
5. **見た目・香り・味わい**をそれぞれ入力
6. 「💾 保存する」で保存

### AI添削の使い方

1. 見た目/香り/味わいのどれかの **「📋 コピー」ボタン**を押す
2. すべての内容が自動でコピーされる（メタデータ＋3項目＋プロンプト）
3. ChatGPT等に貼り付けて送信
4. 受け取った添削結果を **「AI添削結果」欄**に貼り付けて保存

### コピーされる内容

```
ワイン名: ◯◯
タイプ: 赤
産地: ◯◯
品種: ◯◯
ヴィンテージ: 2020
飲んだ日: 2026/06/10

見た目: ...

香り: ...

味わい: ...

このテイスティング表現を10点満点で評価し、改善点を教えてください。
```

```bash
git add index.html README.md
git commit -m "ワインカルテ v1"
git push origin main
```

GitHub リポジトリの Settings → Pages → Source: main branch → Save

公開URL例: `https://mini2kitsune.github.io/wine-karte/`

## データについて

- **保存場所**: ブラウザのlocalStorage（端末内）
- **外部送信なし**: テイスティングデータはサーバーに送信されません
- **写真**: Base64形式でlocalStorageに保存。容量が5MBを超えると警告表示
- **バックアップ**: 大切なデータはブラウザの開発ツール(F12)→Application→LocalStorage から確認できます
